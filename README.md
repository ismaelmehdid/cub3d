## 💡 About the project

> This project is a reproduction of ``wolfenstein 3D`` in ``C`` using ``raycasting``.

### Instructions

**Compiling the game**

To compile, go to the project directory and run :

```shell
$ git submodule update --init --recursive
$ make
```
